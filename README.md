Genshin Impact Fan Landing Page
A visually immersive, single-page fan website dedicated to the open-world RPG, Genshin Impact. This project features a cinematic hero section, game descriptions, lore snippets about the Nations of Teyvat, and social media integration.

Features
Cinematic Header: A full-screen video background featuring the "The Long Way Home" trailer with a dark overlay for text readability.

Sticky Navigation: A smooth menu for navigating between Home, Game Description, Nations (Teyvat), and Social Links.

Responsive Layout: Uses semantic HTML5 tags like <nav>, <section>, and <header> for better structure and SEO.

Interactive UI: Includes a mobile-responsive toggle menu (JavaScript-ready) and external links to official HoYoverse resources.

Gacha & Gameplay Info: A dedicated section explaining the game's mechanics, protagonist (Traveler), and platforms.

Technical Details

Video Background: Utilizes the <video> tag with autoplay, loop, and muted attributes for a seamless cinematic experience.

Anchor Links: Implements ID-based navigation (e.g., #teyvat) to allow users to jump to specific sections of the page.

Accessibility: Includes alt text for images and target="_blank" for external links to ensure a good user experience.

Overlay: A semi-transparent absolute div over the video to ensure text contrast.

Flexbox/Grid: For aligning the "Nations" text and the "Description" image side-by-side.

Typography: Using fonts that match the fantasy aesthetic of the game.

How to Use
Clone the repository.

Ensure your folder structure matches the src paths in the HTML (specifically the ../GamePreview/ directory).

Open index.html in any modern web browser.
